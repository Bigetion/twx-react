# Changelog

## 0.1.2 - 2026-07-19

### Added
- Runtime warning for unsupported utility classes to improve developer experience.
- New getting started guide and quick-start examples.
- Broader utility support for common UI patterns such as `max-w-screen`, `truncate`, `line-clamp`, `sr-only`, and `text-wrap`.

### Improved
- Documentation clarity for installation, quick start, and API selection.
- Test coverage around unknown utilities and sizing behavior.
