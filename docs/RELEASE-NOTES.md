# Release Notes

## 0.1.2

This release focuses on developer experience, documentation clarity, and practical utility support.

### Highlights
- Added runtime warnings for unsupported utility classes.
- Added a new getting started guide for onboarding.
- Expanded support for commonly used utilities such as `max-w-screen`, `truncate`, and `line-clamp`.
- Improved confidence through regression tests for utility support and warnings.

### Suggested next steps
- Review the updated README and getting-started guide.
- Try the examples in the examples folder.
- Share feedback around utility compatibility and ergonomics.
